Projects

May 2024 - Present

Scope: To scan a host system and gather detailed information about its open ports, services, and operating system.

Tool: Nmap (Network Mapper)

Outcome: Conducted an intense scan using the command nmap -T4 -A -v (host IP). The scan results revealed three open ports:

  • Port 135: Microsoft Windows RPC
  • Port 139: Microsoft Windows NetBIOS
  • Port 445: Microsoft SMB

The system was identified as running Microsoft Windows 10/11. After identifying the open ports, I researched potential vulnerabilities and found a critical one (CVE-2020-0796) on Port 135, which was mitigated by closing it using Windows Defender Firewall.

May 2024 - Present

Scope: To identify vulnerabilities present in a host system through thorough scanning.

Tool: Nessus Essentials

Outcome: Conducted an Advanced Scan on a Windows 11 system. The scan revealed 23 vulnerabilities, including a Medium Severity issue (CVSS 5.3) on Port 445, related to SMB Signing Not Required. The vulnerability was mitigated by enabling digital signing in the Local Group Policy Editor, enhancing security.
